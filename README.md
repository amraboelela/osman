# Translate Osman series

Here we show the steps, to download, then translate youtube Osman Episode starting of Episode 90 then you can watch them translated in your computer.

## Setup

```
$ sudo mkdir -p /usr/local/bin
$ sudo curl -L https://yt-dl.org/downloads/latest/youtube-dl -o /usr/local/bin/youtube-dl
$ sudo chmod a+rx /usr/local/bin/youtube-dl
$ vim translation_key.py
key = "<PUT THE KEY FROM YOUR GOOGLE TRANSLATE API ACCOUNT>"
$ sudo easy_install -U requests
$ sudo pip3 install requests
$ sudo pip3 install gTTS
$ /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"
$ brew install ffmpeg
$ brew install handbrake
```
